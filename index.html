Getting started with Behat is as easy as 

vim composer.json

{
    "require": {
        "behat/behat": "2.4@stable"
    },

    "config": {
        "bin-dir": "bin/"
    }
}

curl http://getcomposer.org/installer | php
php composer.phar install
bin/behat --init
vim features/your_first.feature
